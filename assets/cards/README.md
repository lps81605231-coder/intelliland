# 轮播卡片插画

把你的 4 张插画放进**这个文件夹**，文件名必须完全一致（代码按这些路径加载）：

| 文件名 | 对应 Agent |
|--------|-----------|
| `bedtime.jpg`     | 睡前小故事 |
| `memory.jpg`      | 记忆档案馆 |
| `artisan.jpg`     | 知识世界匠人 |
| `picturebook.jpg` | AI 绘本工坊 |

## 尺寸要求
- 比例 **7:10**（竖版），和卡片一致
- 推荐 **840 × 1200 px**（@3x，最清晰），最低 **560 × 800 px**
- 你给的这几张图本身就是 ~1059×1500 的 7:10，直接用即可

## 注意
- 图片**整张**铺满卡片，标题文字叠在**底部约 25% 的留白处**——所以画面主体留在中上部，底部留白别放重要内容（你给的图正好是这样的构图）。
- 格式用 `.jpg`（体积小）。若你的文件是 `.png`，要么导出成 jpg，要么把 `data.js` 里对应的 `img:` 后缀改成 `.png`。
